# Forms Fields
Examples and usage guidelines for form components for creating a wide variety of forms. <br />

Textual form controls, like _input_ and _select_, use the **inputBaseStyles**, found in the _'forms > styles.js'_ file.

<!-- Text input field -->
## Input Field
Be sure to use an appropriate type attribute on all inputs (e.g., email for email address or password for secure password) to take advantage of newer input controls like email verification, password entry, and more.

## Checkbox Field
<Props of={ InputField } />

### Basic use
<Playground>
    <CheckboxField
        label="Checkbox copy"
    />
</Playground>

### With parent component
<Playground>
    {
        class Parent extends PureComponent {
            constructor() {
                super()
                this.state = { checked: false }

                this.handleChange = this.handleChange.bind(this);
            }

            handleChange() {
                const { checked } = this.state;

                this.setState({
                    checked: !checked,
                }, alert(`Changed, checked state is currently ${ checked }`))
            }

            render() {
                return (
                    <CheckboxField
                        label="Checkbox copy"
                        checked={ this.state.checked }
                        onChange={ this.handleChange }
                    />
                )
            }
        }
    }
</Playground>
